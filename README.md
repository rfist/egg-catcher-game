# Egg Catcher Game

A modern web remake of the old electronic game "Nu, pogodi!" (Elektronika IM-02), originally a clone of Nintendo's Game & Watch "Egg" (EG-26).

## Play

Catch falling eggs before they hit the ground. Miss three and it's game over.

**Controls:** `Z` `A` `S` `X` to move between four catch positions, `Space` to start.

## Tech Stack

- React 18 + TypeScript
- Vite
- CSS animations

## Run Locally

```bash
npm install
npm run dev
```
